Portfólio


   Construção de um programa de fidelidade com clusterização de clientes.

   Usei Python, Estatística e técnicas não-supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de performance de compra, a fim de selecionar grupos de clientes para formar um programa de Fidelidade com o objetivo de aumentar a receita da empresa. E o resultado dessa solução, caso fosse implementada, seria de R$ 15MM de dólares de receita anual.

   As ferramentas utilizadas foram:

  • Git, Gitlab e Github
  • Python, Pandas, Matplotlib e Seaborn
  • Jupyter Notebook
  • K-Means, Hierarquical Clustering, DBScan
  • AWS Cloud ( EC2, S3, Postgres,SQLite )
  • Metabase Visualization
